# Your bike images go here

Drop your own photos in **this folder** (`frontend/public/bikes/`) using the exact
filenames below. They will appear automatically on the site after you re-run the
seed (`npm run db:seed` in the `backend` folder).

- Use **JPG** files named exactly `bike-1.jpg` … `bike-12.jpg` (lowercase).
- Add `hero.jpg` for the big image on the home page.
- Recommended size: ~1024×768 (landscape). Any reasonable size works.
- If a file is missing, the site shows a branded gradient placeholder instead of
  a broken image — so it never looks broken, but adding the files makes it look great.

These files are committed to git and deployed with the frontend, so reviewers see
them too.
